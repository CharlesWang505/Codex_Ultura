# 主题包示例

此目录展示 Codex Compass Theme v2 的最小结构。

1. 准备自己的 `wallpaper.png`、`wallpaper.jpg` 或 `wallpaper.webp`。
2. 把图片放入本目录，并让 `theme.json` 的 `wallpaper` 与文件名一致。
3. 如需新任务首页横幅或右侧装饰图，可再加入 `hero.webp`、`portrait.png`，并填写 `showcase.heroImage`、`showcase.portraitImage`。
4. 只压缩目录内的文件，不要额外包一层父目录。
5. 将 ZIP 扩展名保留为 `.zip`，或改为 `.cc-theme`。
6. 在“主题工坊”中点击“导入主题包”。

没有壁纸或展示图片时，可把相应文件名设置为空字符串。Theme v1 主题仍可导入，并会自动迁移。

主题包不得包含脚本、CSS、HTML、可执行文件或远程资源。
